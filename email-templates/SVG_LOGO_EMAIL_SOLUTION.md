### Updated CSS Section (Add to Each Template's <style> tag)

```css
@media (prefers-color-scheme: dark) {
    /* Existing dark mode styles... */
    
    /* SVG Logo Color Changes */
    .logo-text {
        fill: #ffffff !important;
    }
    /* Plus sign stays cyan in both modes */
    .logo-plus {
        fill: #00dbff !important;
    }
}

/* Additional dark mode detection methods */
@media (prefers-color-scheme: dark) {
    .logo-container svg path.logo-text {
        fill: #ffffff !important;
    }
}

/* For Outlook dark mode */
[data-ogsc] .logo-text,
[data-ogsb] .logo-text {
    fill: #ffffff !important;
}

/* Gmail dark mode specific */
u + .email-body .logo-text {
    fill: #ffffff !important;
}
```
